# tela-login
Tela de login desenvolvida com HTML E CSS.
Interface de login desenvolvida como parte dos meus estudos em HTML e CSS.

## Tecnologias usadas

- HTML
- CSS

## Funcionalidades
- Campos de usuário e senha
- Botão de login
- Link para recuperação de senha


## Objetivo
Praticar estruturação de formulários e estilização com CSS, criando uma interface visualmente agradavel.
